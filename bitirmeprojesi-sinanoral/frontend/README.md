## Sürem yetmediği için tamamlayamadım.

To run the app do the following:
* Enter the folder which you downloaded the repository at.
* Open terminal.
* Enter these commands if you are
    * using npm;
        * ``` npm install``` 
        * ``` npm run start``` 
    * using yarn;
        * ``` yarn ``` 
        * ``` yarn start ``` 


#### Technologies/Libraries used
* React
* Typescript
* React Query
* React Router
* Chakra UI
